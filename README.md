# 코루틴 사용해보기
해당 프로젝트는 크게 Auth기능 + 코루틴 + Room 에 초점을 맞추었으나 Auth기능과 Room기능은 다른 파트에서 언급하였으니 코루틴에 초점을 맞추어 작성하였습니다.
***
### :wrench: 기능 설명
- 깃 허브 Auth에 접근하여 로그인을 한다.
- 검색을 통해 깃허브를 사용하는 유저의 레파지토리 정보를 불러온다.
- Room을 사용하여 원하는 레파지토리를 즐겨찾기로 저장한다.
